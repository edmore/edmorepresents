Semester 2 (2013) - Stats

Software and tools

Software

  • Nodes running SLES and capture agents running Ubuntu
  • Nodes run specific Matterhorn 1.4.x (Java) builds i.e. admin build, worker build etc
  • Some agents run the reference (Java) CA software (1.3.x) and our newer CAs run Galicaster 1.3.1 (python)

Sakai - Matterhorn Integration

Monitoring Tools

  • Scripts used to generate our monitoring graphs (Perl)
  • IP Camera dashboard (in Ruby)

Alerts

  • Perl scripts used to notify admins when CAs are down
  • Sintrex emails sent when there are issues with CAs or IP cameras

Deployments

  • Ansible for our new CAs
  • Build script for the nodes

Challenges

  • In venue issues - power / network, audio, venue access
  • Steep-ish Learning curve - tech, workflows
  • User cooperation - lapel mics, consent
  • Workflow Change regression bugs
  • Institutional Environments differ
  • Storage ... backups - ICTs nightmare
